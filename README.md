Python 開発ポートフォリオ
ネットワークの基礎知識（CCNA）と Python による自動化・データ分析スキル

このリポジトリには、ITインフラ運用の効率化や業務自動化を目指して自作した Python アプリケーションを収録しています。

1. 収録アプリケーション一覧
① 野球データ分析・可視化スクリプト (analysis.py)
概要: 大規模な試合結果CSVを読み込み、球団ごとの平均得点を算出してランキングをグラフ表示します。

技術ポイント:

データクレンジング: 不規則な列名や文字エンコーディングに対する堅牢なエラー処理。

Pandas: groupby や to_numeric を用いた高度なデータ集計。

Matplotlib: 分析結果の棒グラフ化（可視化）。

応用: サーバリソースの推移分析や、ネットワークトラフィックの統計レポート作成に応用可能です。

② 本格英単語タイピングソフト (typing_pro.py)
概要: 外部CSVファイルから問題を読み込み、制限時間内にタイピングを行う学習ツール。

技術ポイント:

ファイル入出力: csv.reader を用いた外部データの動的読み込み。

非同期処理: after メソッドによるリアルタイムなタイマー制御。

GUI制御: Tkinter による動的な画面更新とスコア管理。

③ GUI版 数あてゲーム (guess_number.py)
概要: 1〜100の数字を当てるインタラクティブなゲーム。

技術ポイント:

状態管理: ゲーム状況に応じたボタンの有効/無効化（state="disabled"）制御。

ユーザー体験: 入力欄の自動クリアや色による判定結果の強調。

④ タイピング練習・シンプル版 (typing_basic.py)
概要: 基本的なタイピング判定とイベント処理の学習用ツール。

技術ポイント:

イベントバインド: <Return>（Enterキー）によるシームレスな操作性の実現。

2. 使用技術スタック

Tkinter (デスクトップアプリ開発)
