CCNAの知識とPythonを掛け合わせ、実務の効率化を目指して自作したツール群です。

## 1. 収録アプリ一覧
- **英単語タイピングソフト（本格版）**: CSV連携、タイマー処理、スコア判定を実装。
- **数あてゲーム**: GUIの状態管理（ボタンの有効/無効化）を学習。
- **タイピング練習ツール（シンプル版）**: イベントバインド（Enterキー判定）の基礎。

## 2. 使用技術
- **言語**: Python 3.x
- **ライブラリ**: Tkinter (GUI), csv (データ操作), random
- **技術要素**: 例外処理（try-except）、非同期処理（afterメソッド）、ファイル入出力

## 3. インフラエンジニアとしての視点
CCNAで学んだ「正確な手順の実行」と「ログ解析」の重要性を背景に、
将来的にネットワーク機器の自動設定やログ監視ツールを開発するための基礎技術を習得しています# python-gui-apps
